# Panorama Control

`BmPanorama`

## Instance Properties

|name|type|default|description|
|------|-----|-----|----|
|anchor|ControlAnchor||The location of the control.|
|offset|Size||The offset of the control.|

## Examples

### Add a panorama control on the map

#### Code

```html
<template>
  <baidu-map class="map" center="北京">
    <bm-panorama></bm-panorama>
  </baidu-map>
</template>
```
